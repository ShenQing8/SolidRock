# 3D 第三人称塔防游戏 Demo

一款基于 Unity 开发的 3D 塔防游戏原型。玩家控制角色在场景中移动、建造防御塔，抵御敌人进攻并保护主塔。项目采用 MVC 架构，使用 JSON 进行数据配置，并应用了多种设计模式（单例、对象池等）来优化性能和代码结构。

## 功能特点

- **第三人称角色控制**：流畅的移动、攻击动画，基于 Unity 动画状态机。
- **塔防核心玩法**：建造防御塔，自动攻击敌人；敌人通过 NavMesh 寻路。
- **数据驱动设计**：怪物波次、塔属性等通过 JSON 配置，可随时调整。
- **性能优化**：使用对象池管理敌人，减少运行时 GC 开销。
- **清晰的架构**：基于 MVC 模式分离数据、视图和逻辑，易于扩展。
